# 📚 Attendify

A modern full-stack **Attendance Tracking System** built with the **MERN Stack** that helps students manage and analyze their attendance efficiently. Attendify provides a clean dashboard, subject-wise attendance tracking, timetable management, calendar view, and attendance analytics.

🌐 **Live Demo:** https://attendify-six-beryl.vercel.app/

---

## ✨ Features

- 🔐 JWT Authentication
- 📊 Dashboard with attendance overview
- 📖 Subject-wise attendance management
- ✅ Mark Present / Absent / No Class
- 📅 Timetable Management
- 🗓️ Calendar View
- 📈 Attendance Analytics
- ⚙️ User Settings
- 📱 Fully Responsive UI
- 🍪 Secure Cookie-based Authentication

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router
- Axios
- Lucide React
- React Hot Toast

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Cookie Parser
- bcrypt.js

### Deployment
- Frontend: Vercel
- Backend: Vercel
- Database: MongoDB Atlas
